Инструменты разработчика

При написании кода всегда будут оповещения и ошибки, это нормально. Консоль браузера показывает информацию связанную с веб-страницей, в том числе сообщения об ошибках и предупреждения исполнения JS-кода, запущенного в контексте страницы. Следующие комбинации клавиш открывают инструменты разработчика на вкладке Console: Windows и Linux - Ctrl + Shift + J или F12 MacOS - Command + Option + J Дополнительные материалы Вывод сообщений в консоль https://developer.chrome.com/docs/devtools/console/log/ Справочник по API консоли https://developer.chrome.com/docs/devtools/console/api/
